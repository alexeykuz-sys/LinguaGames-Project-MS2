Rules:
1. Start the game by choosing the language you would like to practice.
2. PRess Start button.
3. Start flipping cards one by one.
3. You have to match english word to the word in the language of your choice.
4. If you can't find a pair, the flipped cards will be flipped back with the face down. You have one minute to complete the round. If you wish to restart the game, press reset button and start again.
