Rules: Start the game by choosing the language you would like to practice. Then start flipping a card. You have to match english word to the same word in the language of your choice. If you can't find a pair, the flipped cards will be flipped back with the face down. You have one minute to complete the round. If you wish to restart the game, press reset button and start again.
